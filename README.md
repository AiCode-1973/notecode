# Notepad System Integration Plan

Building a premium, modern notepad system with a focus on aesthetics and user experience.

## Tech Stack
-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
-   **Backend**: PHP (XAMPP environment)
-   **Database**: SQLite (Zero-config, portable)

## Features
-   [ ] **Authentication**: Secure Login and Registration screens.
-   [ ] **Dashboard**: Sidebar navigation with "All Notes" and "Favorites".
-   [ ] **Notes Management**: Create, Read, Update, Delete with Titulo and Descricao.
-   [ ] **Accents**: Favorites toggling.
-   [ ] **Visuals**: Glassmorphism, smooth animations, dark-themed premium design.

## Structure
-   `index.php`: Dashboard entry.
-   `login.php`: Login interface.
-   `register.php`: Registration interface.
-   `api/notes.php`: Backend logic for notes.
-   `api/auth.php`: Backend logic for authentication.
-   `assets/css/style.css`: Core design system.
-   `assets/js/app.js`: Frontend interactions.
